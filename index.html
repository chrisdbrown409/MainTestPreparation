Test Preparation

Test preparation is the step in software development where testers design and create testing documentation. Generally this step is done in parrellel while developers are making the code changes. In some instances, test cases are written before development (e.g. Test Driven Development). The methodology can impact or dicate the type of documentation that is needed.

While creating documentation may take time, there are many benefits to these steps as the testers are given the opportunity to share with the team how the development changes will be tested and gives the team the opportunity to dicuss the testing approach and consider gaps or missed requirements.
