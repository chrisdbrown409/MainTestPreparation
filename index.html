Test Case Management

Test Case Management tools help testers build, track, maintain and reuse test case information.

There are many tools available for this purpose. The following tools are just a few examples of the options available. Many of these tools also directly integrate with issue tracking software (e.g. Jira) or automation tools. It is important to research all the options that are available before selecting which tool best meets the needs but staying within the alloted budget.
